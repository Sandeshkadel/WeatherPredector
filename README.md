## Overview

Welcome to **WeatherPredector**, a modern, interactive, and fully branded weather prediction app. 
WeatherPredector not only provides accurate weather forecasts but also delivers an engaging, visually appealing, and user-friendly experience.  
This app was designed as a **learning-focused frontend project**, showcasing real-time weather data, dynamic UI updates, and immersive visuals.

---

## Features

- **Responsive Design:** Optimized for laptops and tablets, maintaining consistent layout across screen sizes.  
- **Dynamic Weather Icons & Backgrounds:** Weather icons and background visuals update based on current conditions.  
- **Hourly & Weekly Forecasts:** Plan your day or week with real-time weather data for any location.  
- **Today's Highlights:** UV Index, Wind Speed, Humidity, Visibility, Air Quality, Sunrise & Sunset.  
- **Search Functionality:** Easily search for weather updates in any city worldwide.  
- **Weather-Based Ambient Sound (Optional):** Calms ambient sound for clear weather, rain, or snow. Users can enable or disable sounds manually.  
- **Animated Preloader:** Smooth Lottie animation while data is loading, enhancing user experience.
